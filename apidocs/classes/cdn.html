cdn Class

Uses
  • utils
Defined in: lib/cdn.js:23
Module: cdn
Parent Module: yui

The yui.cdn extension that provides some basic configuration that will facilitate the configuration of YUI to be served from CDN.

When the "cdn" sub-module is used by yui middleware, it will allow you to explicitly ask for booting YUI from YUI CDN in the client runtime. In the following example you can see how to force yui to load from CDN. This is the default behavior, but in some cases, when defining a custom base and root for all groups you might want to set YUI straight to the original YUI CDN.

// Creates a new express app.
var app = express();
app.yui.setCoreFromCDN();
