getGroupConfig

(
  • path
)
Object protected

Defined in lib/groups.js:46

Extracts information from a meta module. This information includes the details about the meta module in the YUI registry plus all the info about the groups that are part of the meta module, along with all the modules per group. This method is used by origin to analyze each group before serving them.

var path,
    config;
path = __dirname + '/assets/js/metas.js';
config = app.yui.groups.getGroupConfig(path);

Parameters:

  • path String

    the filesystem path to the meta module

Returns:

Object: modules the modules meta for the group
